Place your input videos in this folder for experiment runs.

Recommended formats:
- `.mp4` with H.264 encoding
- `.avi`
- `.mov`
- `.mkv`

Recommended video characteristics:
- Resolution: `640x480` or `1280x720`
- Frame rate: `24-30 FPS`
- Duration: `30-90 seconds`
- Content: moving people, cars, or everyday scenes with visible objects

Recommended file set:
- `cam1.mp4`
- `cam2.mp4`
- `cam3.mp4`
- `cam4.mp4`

Nice-to-have for the 8-stream experiment:
- `cam5.mp4`
- `cam6.mp4`
- `cam7.mp4`
- `cam8.mp4`

If you only have 1-4 files, the experiment runner will automatically reuse them for higher stream counts.
